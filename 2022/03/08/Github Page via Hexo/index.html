通过Github Page和Hexo搭建个人博客

重要命令

npm命令安装Hexo

npm install -g hexo-cli

初始化博客（创建blog文件夹）

hexo init blog

启动服务（localhost:4000）

hexo s

配置站点信息

deploy:
    type: git
    repo: git@github.com:missingstuffedbun/missingstuffedbun.github.io.git
    branch: main
url: https://missingstuffedbun.github.io/

部署到站点（Github Page）

hexo d

Debug

ERROR Deployer not found: git

安装hexo-deployer-git

npm install --save hexo-deployer-git
