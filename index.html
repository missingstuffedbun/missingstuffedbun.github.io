启动Grafana

docker stop grafana & docker rm grafana
docker run -d --name grafana -p 3000:3000 grafana/grafana

基本信息

公网IP： 47.107.123.109
虚拟网卡（可以通过ifconfig查询）
Grafana IP： 172.17.0.2
MySQL IP： 172.17.0.1

修改MySQL配置

vi /etc/mysql/mysql.conf.d/mysqld.cnf

bind-address = 0.0.0.0

新用户并授权（注意区分符号’和`）

CREATE USER 'remote'@'172.17.0.2' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON `bus`.* TO `remote`@`172.17.0.2` WITH GRANT OPTION;
FLUSH PRIVILEGES;

查询用户及授权

update user set host='localhost' where user='root';
select user,host from user;
show grants for remote@'172.17.0.2';

重启MySQL

systemctl restart mysql

建表

地理位置表

create table t_geo  select id, gps_code, gps_direction from t_bus_status;

路线表

create table t_road select distinct road_id, road_name, site_id, site_no, site_name from t_bus_status;

展示在Grafana

线路的车次

SELECT
  $__timeGroupAlias(gps_date,$__interval),
  road_id AS metric,
  count(site_id) AS "site_id"
FROM t_bus_status
WHERE
  $__timeFilter(gps_date)
GROUP BY 1,2
ORDER BY $__timeGroup(gps_date,$__interval)
